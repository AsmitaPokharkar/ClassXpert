# Unified Learning Platform - ClassXpert

A modern, feature-rich learning management system frontend built with Vite, React, TypeScript, and Tailwind CSS. This platform provides an intuitive interface for course management, analytics dashboards, content creation, and more. This was developed by me along with my groupmates as a Sem 3 Mini Project.
For live demo : https://unifiedlearningplatform.netlify.app/

##  Features :

-  **React 18** with TypeScript for robust type safety
-  **Vite** for fast development and optimized builds
-  **Tailwind CSS** with custom theming (light/dark mode support)
-  **Recharts** for interactive data visualizations
-  **React Quill** rich text editor for content creation
-  **Learning analytics** dashboard
-  **Toast notifications** with react-hot-toast
-  **React Router DOM** for seamless navigation
-  **Framer Motion** for smooth animations
-  **Date-fns** for date manipulations
-  **React Markdown** for markdown rendering
-  **Lucide React** for beautiful icons
-  **ESLint** + TypeScript ESLint for code quality
-  Dark mode support out of the box

##  Tech Stack :

- **Frontend Framework**: React 18
- **Build Tool**: Vite 5
- **Language**: TypeScript
- **Styling**: Tailwind CSS 3
- **Routing**: React Router DOM 6
- **Charts**: Recharts 2
- **Rich Text Editor**: React Quill 2
- **Animations**: Framer Motion 11
- **Icons**: Lucide React
- **Notifications**: React Hot Toast
- **Markdown**: React Markdown 9
- **Date Utility**: Date-fns 3
- **UUID**: UUID 9

##  Installation :

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/unified-learning-platform.git
   cd unified-learning-platform
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```

##  Available Scripts :

| Script        | Description                                   |
| ------------- | --------------------------------------------- |
| `npm run dev` | Starts the Vite development server            |
| `npm run build` | Builds the app for production to `dist/`   |
| `npm run lint`  | Runs ESLint on the project                   |
| `npm run preview` | Locally previews the production build      |
